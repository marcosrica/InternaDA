# VueJS

Vue es un framework de javascript (y typescript) que se suele usar en conjunto con node y express. Tiende a ser muy flexible en su organización, permitiendo abstracciones y una especie de construcción a base de "bloques de lego" que lo hace muy sencillo de mantener. Es el que se usó, por ejemplo, para construir [MejoraDA](/servicios/mejorada).
